# Adaptrix Developer Hiring Challenges 2026

This repository contains evaluation challenges for Junior and Senior developer candidates.

## Challenges

### [Junior Challenge](junior/)
**Entity Extraction Pipeline Basics**
- Time: 30 minutes
- Stack: Python, LLM API
- Task: Build a document processing pipeline that extracts KPIs, dates, and organizations from business reports

### [Senior Challenge](senior/)
**PDF Bounds Matching for Combined Entities**
- Time: 30 minutes
- Stack: Python + TypeScript
- Task: Implement sophisticated text matching strategies for entities that don't exist as literal text in PDFs

## Getting Started

1. Clone this repository
2. Navigate to the appropriate challenge folder
3. Follow the README instructions in that folder

## Requirements

- **Both Challenges**: LLM API key (OpenAI, Anthropic, etc.)
- **Junior**: Python 3.10+
- **Senior**: Python 3.10+ and Node.js 18+

## AI Assistance

These challenges are designed to be completed with AI assistance. Use Claude, ChatGPT, Copilot, or any other AI tools to help you code faster and solve problems efficiently.
